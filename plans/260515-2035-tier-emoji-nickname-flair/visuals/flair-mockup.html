5. Edge cases hiển thị
Case Input Output
Nick dài SuperLongNicknameThatIsThirtyChar (32) + 🟡 SuperLongNicknameThatIsThi 🟡
Nick có emoji cũ Tohma ⚫ → lên tier Đồng Tohma 🟤 (strip ⚫ rồi append 🟤)
Server owner Owner lên tier Skip + log warn (Discord không cho)
User opt-out + lên tier flair_enabled = 0 Không touch nickname
⚠️ Lưu ý: Đây là mockup HTML, emoji render trên Discord client có thể khác chút (kích thước, anti-alias) tùy theme dark/light và OS.
